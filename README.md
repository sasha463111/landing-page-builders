# דף נחיתה - SmartLeads

דף נחיתה מקצועי בעברית עם עיצוב מודרני ורספונסיבי.

## תכונות

- ✨ עיצוב מודרני עם צבעי נגיעה ירוקים (#34D399)
- 📱 רספונסיבי מלא (Mobile, Tablet, Desktop)
- 🎨 פונט Heebo עם משקלים שונים
- 🔄 גלגלים אינסופיים עם אנימציות
- 📝 טופס יצירת קשר עם ולידציה
- ⚡ אנימציות scroll וחווית משתמש חלקה
- 🌐 תמיכה מלאה בעברית (RTL)

## מבנה הקבצים

```
landing-page/
├── index.html      # מבנה HTML של הדף
├── styles.css      # כל העיצובים והסטיילים
├── script.js       # JavaScript לאנימציות ופונקציונליות
└── README.md       # קובץ זה
```

## התקנה והפעלה

### שיטה 1: פתיחה ישירה
פתח את הקובץ `index.html` בדפדפן שלך.

### שיטה 2: שרת מקומי
אם יש לך Python מותקן:
```bash
cd landing-page
python3 -m http.server 8000
```
ואז פתח בדפדפן: `http://localhost:8000`

או עם Node.js (אם יש לך http-server):
```bash
cd landing-page
npx http-server
```

## עיצוב

### צבעים
- **ירוק ראשי**: `#34D399`
- **רקע שחור**: `#000000`
- **רקע כהה**: `#111111`
- **טקסט לבן**: `#ffffff`
- **טקסט אפור**: `#e5e7eb`
- **אדום להדגשה**: `#ef4444`

### טיפוגרפיה
- **פונט**: Heebo (מ-Google Fonts)
- **משקלים**: 300, 400, 500, 700, 900

### מרווחים
- מרווחים נדיבים בין סעיפים
- Padding ומרווחים מותאמים לגדלי מסך שונים
- Breakpoints: 480px (Mobile), 768px (Tablet), 1400px (Desktop)

## סעיפי הדף

1. **Header** - לוגו וראש הדף
2. **Hero Section** - כותרת ראשית, סרטון, וקישור לפגישה
3. **Stats Section** - נתונים והצלחות (100+ לקוחות, 7x החזר, וכו')
4. **Problem Section** - תיאור הבעיות הנפוצות
5. **Solution Section** - הפתרון המוצע
6. **Testimonials** - עדויות לקוחות וגלגלים אינסופיים
7. **About Section** - מידע על החברה
8. **Team Section** - הצוות
9. **FAQ Section** - שאלות נפוצות
10. **Contact Form** - טופס יצירת קשר
11. **Footer** - קישורים ותקנון

## אנימציות

- **גלגלים אינסופיים**: גלגל ירוק (חיובי) וגלגל אדום (שלילי) מסתובבים ברציפות
- **Scroll Animations**: אלמנטים מופיעים בהדרגה תוך כדי גלילה
- **Stats Counter**: מספרים עולים בהדרגה כשחלק ה-Stats נכנס לתצוגה
- **Hover Effects**: אפקטים על כפתורים וקישורים

## התאמה אישית

### שינוי צבעים
ערוך את המשתנים ב-`styles.css`:
```css
:root {
    --primary-green: #34D399;
    --bg-black: #000000;
    /* ... */
}
```

### הוספת תוכן
ערוך את הקובץ `index.html` והוסף תוכן חדש בתוך הסעיפים המתאימים.

### שינוי אנימציות
ערוך את קובץ `script.js` כדי להתאים את מהירות האנימציות והתנהגותן.

## הערות חשובות

1. **טופס יצירת קשר**: כרגע הטופס מדמה שליחה. יש לחבר לשרת אמיתי במקום הפונקציה ב-`script.js`.

2. **סרטון**: ה-placeholder לסרטון זקוק להחלפה בווידאו אמיתי או modal.

3. **תמונות צוות**: כרגע יש placeholders. הוסף תמונות אמיתיות.

4. **Google Fonts**: הפונט Heebo נטען מ-Google Fonts. ודא שיש חיבור לאינטרנט.
