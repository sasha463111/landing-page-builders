# תמונות נדרשות

אנא הוסף את הקבצים הבאים לתיקיית `public/`:

## תמונות נדרשות:

1. **`logo-builders.png`** - לוגו של Builders
   - גודל מומלץ: 400x200px או יותר
   - פורמט: PNG עם רקע שקוף או לבן

2. **`sasha-dibka.jpg`** - תמונה של סשה דיבקה
   - גודל מומלץ: 500x500px או יותר (ריבוע)
   - פורמט: JPG או PNG

## תמונות לקוחות (אופציונלי - כרגע יש placeholders):

אם יש לך תמונות אמיתיות, תוכל להחליף את ה-placeholders ב-`Hero.jsx`:

- עומרי כהן
- מתן ניסטור  
- קבוצת בסר
- דניאל מולדבסקי
- עורך דין אילן

**הערה**: כרגע התמונות מוגדרות עם placeholder אוטומטי מ-via.placeholder.com. 
אם תשיים תמונות בתיקיית `public/`, עדכן את ה-`src` ב-`Hero.jsx` ל-`/filename.jpg`
